2. Edit Outcome Drawer
Satır tıklamasında veya "Edit Outcome" seçildiğinde açılan drawer
1 Header: Code badge (mono font, gradient) + "Edit Outcome" title. Code ve label locked — sadece drawer üzerinden değiştirilmez (readonly, ikonu ile).
2 Body: Üç section — Identity (readonly code+label), Description (editable textarea), Criterion Mapping (pill selector + coverage type toggle).
3 Coverage Type: Direct/Indirect radio seçimi — her mapping için hangi seviyede katkı sağladığı belirtilir. Mevcut sistemdeki "cycle on click" yerine açık seçim.
