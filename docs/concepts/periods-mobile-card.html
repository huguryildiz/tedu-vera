Variant B — Status Medallion (countdown/badge hybrid)

Design notes

  • Variant A mirrors the Projects card pattern exactly: ring / title / kebab hero → meta row → config strip → footer stats. Ring color changes per state (indigo · green · slate). Metric is adaptive — setup % for draft, eval % for live, 100% for closed.
  • Variant B uses a squared medallion instead of a ring. Draft shows a countdown (“10 DAYS”), live shows a mini % tile with pulse dot, closed shows a lock icon. More informationally varied but breaks the Projects symmetry.
  • Both variants collapse the sparse eyebrow-labeled rows from the current layout into a dense config strip where criteria + outcome sit side by side (saves ~60px vertical).
  • Date range moves into a meta row with the status pill, removing its own labeled block.
  • Readiness blockers surface as a compact 2 blockers pill next to the Draft pill instead of a separate Ready/Blocked row.
  • Footer keeps projects · jurors · updated consistent with Projects for scannability.
