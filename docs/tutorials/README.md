# VERA Product Tour

VERA is a multi-tenant academic jury evaluation platform built for universities that need a defensible, accreditation-ready record of capstone project assessments. Admins configure evaluation periods with custom rubrics mapped to ABET, MÜDEK, or institutional outcomes; jurors score projects from any device during a live jury day; and the platform generates rankings, outcome-attainment analytics, and a tamper-evident audit log — all without requiring any spreadsheet work.

The screenshots in these walkthroughs were captured automatically from the live demo environment (TEDU — Electrical-Electronics Engineering, Spring 2026) using Playwright. They reflect the actual running application.
